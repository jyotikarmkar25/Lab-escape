## 🎮 Game Overview

**Lab Escape** is a fast-paced infinite runner where you control a scientist escaping from a dangerous chemistry lab.

Avoid obstacles, survive as long as possible, and beat your high score.

---

## 🎯 Controls

- ⬅️ ➡️ **Move Left / Right**
- ⬆️ **Jump**
- ⬇️ **Slide** *(planned feature)*

---

## ✨ Current Features

- Smooth 60 FPS gameplay loop  
- Player movement (left, right, jump)  
- Obstacle spawning system  
- Collision detection  
- Real-time score tracking  
- Responsive canvas (auto resize)  
- Neon-themed UI with animations  
- Start screen & Game Over screen  
- High score stored using localStorage

## 🚀 Getting Started

1. Download or clone this project  
2. Open `index.html` in your browser  
3. Click **PLAY** and start the game  

No installation or setup required.
